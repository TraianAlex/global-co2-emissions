# CO2 Emissions By Year React App

Designed by picture resources provided.

Configurable params - MIN_YEAR, MAX_YEAR, DEFAULT_YEAR are in `src/app/utils.ts`.

The `marks` prop for Slider is dynamic. The slider can be changed by modifying the `MIN_YEAR` value and it will show more intervals while stretching automatically.

## Scripts

- `npm run dev` - Start local development server.
- `npm run build` - Type-check and create production build.
- `npm run test` - Run test suite with Vitest.
